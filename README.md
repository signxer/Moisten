---

有些事，不必亲力亲为。

Moisten 在你看不见的地方，安静地替你完成那些重复的、耗时的、却又不得不做的事情。你只管去做更重要的事。

### 特性

- 后台静默运行，无需值守
- 多线程并发，效率至上
- 断点续传，中途退出亦可继续
- 目标导向，达成即止

### 使用

前往 [下载页面](https://signxer.github.io/Moisten/) 获取最新版本。

首次运行需安装环境：

```bash
pip install -r requirements.txt
python -m playwright install chromium
```
